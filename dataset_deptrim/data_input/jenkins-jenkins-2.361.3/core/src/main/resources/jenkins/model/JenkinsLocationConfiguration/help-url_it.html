Specificare facoltativamente l'indirizzo HTTP dell'installazione di Jenkins, come http://host.dominio/jenkins/ . Questo valore è utilizzato per far sapere a Jenkins come far riferimento a se stesso, ad es. per visualizzare immagini o creare collegamenti nei messaggi di posta elettronica.

Ciò è necessario perché Jenkins non può rilevare autonomamente tale URL in modo affidabile.
