Arquiva os artefatos de construção (por exemplo, arquivos zip de distribuição ou arquivos jar) desta forma eles podem ser baixados mais tarde. Arquivos armazenados serão acessíveis da página web do Jenkins.
Normalmente, o Jenkins mantém os artefatos para uma construção tanto tempo quanto um log de construção é mantido, mas se você não necessita de artefatos antigos e gostaria de economizar espaço em disco, é possível apagá-los.
