コンピュータの仮想メモリ容量(一般的には"スワップ容量")を監視し、閾値を下回るようであればそのエージェントをオフラインにします。

スワップ容量の正確な定義はプラットホーム依存です。

  • Windowsにおけるこの値は、 ページファイルでの利用可能な容量 ですが、 Windowsはページファイルサイズを自動的に拡張できるので、値にあまり意味はありません。
  • Linuxでのこの値は、 /proc/meminfo から取得します。
  • 他のUnixシステムでのこの値は、 top コマンドの実行結果から取得します。

もし、この値が報告されないOSがあれば、改善できるように連絡してください。
