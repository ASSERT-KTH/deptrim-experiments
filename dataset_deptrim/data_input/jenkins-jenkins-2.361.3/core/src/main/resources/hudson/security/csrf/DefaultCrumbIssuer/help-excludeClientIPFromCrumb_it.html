Alcuni proxy HTTP filtrano le informazioni che il componente predefinito di emissione dei crumb utilizza per calcolare il valore del nonce. Se c'è un proxy HTTP collocato fra il browser e il server Jenkins e si riceve una risposta 403 quando si invia un modulo a Jenkins, l'abilitazione di quest'opzione potrebbe essere di aiuto. L'utilizzo di quest'opzione rende il valore del nonce più facile da falsificare.
