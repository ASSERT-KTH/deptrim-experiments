Archivia gli artefatti di compilazione (ad esempio, file ZIP per la distribuzione o file JAR) in modo da consentirne il download in un secondo momento. I file archiviati saranno accessibili dall'interfaccia Web di Jenkins.
Normalmente, Jenkins mantiene gli artefatti di una compilazione fintantoché viene mantenuto il log di tale compilazione, ma se i vecchi artefatti non sono necessari e se si preferirebbe risparmiare spazio su disco, è possibile farlo.

Si noti che il tipo di processo Maven archivia automaticamente tutti gli artefatti Maven prodotti. Gli artefatti configurati qui saranno archiviati in aggiunta a questi. L'archiviazione automatica degli artefatti può essere disabilitata dalle opzioni avanzate di Maven.
